Exemple du lien tuberculose - antituberculeux (Cf image):
- Wikidata décrit les molécules traitant la tuberculose (pyrazinamide, éthambutol...). 
- Ces molécules sont commercialisées (noms commerciaux des médicaments: dexambutol, myambutol...)
- Certains médicaments sont achetés par le CHU de Bordeaux (code UCD 3400890258982 - code NIMED 4933)

Quand le clinicien recherche le mot "tuberculose" dans le moteur de recherche : 
1. Identification que "tuberculose" est une maladie décrite dans le graphe de connaissance
2. Recherche de liens avec d'autres concepts médicaux dans le graphe de connaissance (médicaments...) pour identifier quelles données structurées et non structurées seront à rechercher/recommander
3. Réponse à l'interface qu'elle doit rechercher/recommander:
    - Dans les données structurées: liste de codes NIMED (prescription/administration d'un antituberculeux durant une hospitalisation)
    - Dans les données semi-structurées (DxCare) et en texte libre: liste des termes (molécules, noms commericaux, famille de traitement...)
